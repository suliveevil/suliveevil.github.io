## zsh plugin manager

### [Antigen](https://github.com/zsh-users/antigen)

&#160;&#160;&#160;&#160;The plugin manager for zsh.（Latest commit c91f77c on 2018-01-16）
&#160;&#160;&#160;&#160;Antigen is a small set of functions that help you easily manage your shell (zsh) plugins, called bundles. The concept is pretty much the same as bundles in a typical vim+pathogen setup. Antigen is to zsh, what Vundle is to vim.

### [oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh)

&#160;&#160;&#160;&#160;A delightful community-driven (with 1,200+ contributors) framework for managing your zsh configuration. Includes 200+ optional plugins (rails, git, OSX, hub, capistrano, brew, ant, php, python, etc), over 140 themes to spice up your morning, and an auto-update tool so that makes it easy to keep up with the latest updates from the community.
